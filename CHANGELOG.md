# Changelog

## v1.2.0 — 2026-03-30

### 보안
- SECRET_KEY 기본값 사용 시 경고 추가
- 가족 IDOR 수정 (family_id 소유권 검증)
- 알림 설정 엔드포인트에 인증 추가
- Gemini API rate limiting 추가 (slowapi, 환경변수로 조절 가능)
- 비밀번호 최소 8자 정책 적용
- 식품안전나라 API HTTP→HTTPS 변경

### 추가
- 가족 마스터를 master_id 필드로 명시적 관리
- 식재료별 추이 탭에서 검색어 없을 때 전체 식재료 목록 표시
- 인벤토리에서 식재료 클릭 시 수정 모달 (이름, 가격, 매장, 유통기한)
- 직접등록 시 Gemini normalizer 사용 (브랜드명 정확한 제거)
- 회원가입 비밀번호 검증 에러 한국어 메시지 표시
- Gemini 모델 fallback 체인 (2.5-flash → 2.0-flash 자동 전환)
- 스캔 503 에러 시 자동 재시도 (최대 2회, exponential backoff)

### 수정
- 로그인 사용자에 따라 마스터가 달라지던 문제 해결
- 공동 편집 설정을 마스터만 변경 가능하도록 제한
- 가격 검색 시 부분 매칭으로 엉뚱한 결과가 나오던 문제 해결
- 매장별 비교가 가격 이력과 불일치하던 문제 해결
- 프론트엔드 비밀번호 minLength 6→8 백엔드와 일치
- Safari 호환성 (개발환경 쿠키 secure=false)
- 스캔 에러 시 Gemini 내부 에러 대신 한국어 메시지 표시
- 식재료 수정 시 이름 변경되면 normalized_name 재생성
- 3자 이하 이름은 Gemini 호출 스킵 (API 호출 절약)
- 마스터 탈퇴/이동 시 남은 멤버 중 가장 오래된 사람에게 마스터 자동 이전

## v1.1.0 — 2026-03-30

### 새로운 기능
- **Gemini 영수증 스캔** — 영수증 촬영 한 번에 식재료/가격/매장명 자동 인식. 세제, 생활용품 자동 제외
- **AI 레시피 추천** — 식품안전나라 API + Gemini AI 보충. 냉장고 재료 기반 맞춤 추천
- **가계부** — 월별 지출 바 차트, 식재료별 가격 추이, 매장별 비교, 인플레이션 알림
- **나만의 레시피** — 직접 레시피 등록/수정/삭제, 사진 업로드, 냉장고 재료 매칭 시 추천에 포함
- **가족 구성원 내보내기** — 마스터가 구성원을 내보내기 가능
- **월별 예산 설정** — 예산 초과 시 경고, 냉장고 화면에 프로그레스 바

### 개선
- 직접 등록에 가격/매장명/수량(자유 텍스트) 필드 추가
- 레시피 출처 표시 (나의 레시피 / 식품안전나라 / AI 추천)
- 식재료 이름 정규화 (베지밀B → 두유, 냉동 삼겹살과 생삼겹살 구분)
- 가계부 식재료 검색 자동완성 (구매 이력 기반)

### 버그 수정
- 직접 등록 실패 (quantity 타입 불일치)
- 모달이 네비/헤더에 가려지는 문제
- 레시피 매칭 루프 break 버그
- 등록 시 Gemini 동기 호출로 인한 지연
- 스캔 결과 모달 스크롤 안 되는 문제

### 디자인
- 터치 타겟 44px 최소 보장
- No-Line 규칙 적용 (구분선 제거)
- 섹션 제목 크기 개선
- 스캔 페이지 카메라 영역 축소
