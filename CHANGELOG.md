# Changelog

## v1.4.0 — 2026-03-31

### 추가
- **PWA 설치 지원** — 홈화면에 앱 아이콘 추가 가능 (manifest, 아이콘, Apple 메타태그)
- Service Worker에 fetch 핸들러 추가 (Chrome PWA 설치 조건 충족)
- VAPID 키 설정으로 푸시 알림 활성화

### 개선
- 전체 프로젝트 "유통기한" → "소비기한" 용어 통일 (프론트엔드, 백엔드, 문서)
- 회원가입 페이지 문구 간소화
- 설정 페이지 푸시 알림 꺼짐 상태에서 불필요한 설명 문구 제거

## v1.3.0 — 2026-03-31

### 추가
- **푸시 알림** — Web Push API로 소비기한 알림을 기기에 직접 전송. 설정 페이지에서 ON/OFF 가능
- **알림 스케줄러** — APScheduler가 15분 간격으로 만료 체크 + 푸시 전송 (설정한 알림 시간에 맞춰 발송)
- 알림 설정이 가족별로 분리됨 (이전: 전체 공유 → 이제: 가족마다 독립 설정)
- 설정 페이지에 **푸시 시간 선택기** 추가 (30분 단위)
- 설정 페이지에 **알림 주기 토글** 컴포넌트 신규 구현
- 식재료 목록 API에 페이지네이션 지원 (`limit`, `offset` 파라미터, 기존 응답 하위 호환)
- 알림 로그 API 페이지네이션 (`limit=20`, `offset`)
- 알림 페이지에 "더 보기" 버튼
- Service Worker 등록 (푸시 수신 전용, 캐싱 없음)
- iOS Safari에서 PWA 설치 안내 메시지 표시
- `.env.example`에 VAPID 키 항목 + 생성 방법 추가

### 성능
- DB 커넥션 풀 설정 (`pool_size=10`, `max_overflow=20`, `pool_pre_ping`)
- 식재료/알림 로그 테이블에 복합 인덱스 추가
- 만료 체커를 벌크 쿼리로 최적화 (N*M 개별 쿼리 → 가족당 1회 벌크)
- 만료 중복 체크를 LIKE 패턴 → `days_before` 정확 매칭으로 변경

### 보안
- 알림 설정 IDOR 수정 — 다른 가족의 설정을 수정할 수 없도록 `family_id` 소유권 검증 추가
- VAPID 키 미설정 시 서버가 503 반환 (앱 시작은 허용, 경고 로그)

### 수정
- 설정 페이지 버전 v1.2.0 → v1.3.0 업데이트

## v1.2.0 — 2026-03-30

### 보안
- SECRET_KEY 기본값 사용 시 경고 추가
- 가족 IDOR 수정 (family_id 소유권 검증)
- 알림 설정 엔드포인트에 인증 추가
- Gemini API rate limiting 추가 (slowapi, 환경변수로 조절 가능)
- 비밀번호 최소 8자 정책 적용
- 식품안전나라 API HTTP→HTTPS 변경

### 추가
- 가족 마스터를 master_id 필드로 명시적 관리
- 식재료별 추이 탭에서 검색어 없을 때 전체 식재료 목록 표시
- 인벤토리에서 식재료 클릭 시 수정 모달 (이름, 가격, 매장, 소비기한)
- 직접등록 시 Gemini normalizer 사용 (브랜드명 정확한 제거)
- 회원가입 비밀번호 검증 에러 한국어 메시지 표시
- Gemini 모델 fallback 체인 (2.5-flash → 2.0-flash 자동 전환)
- 스캔 503 에러 시 자동 재시도 (최대 2회, exponential backoff)

### 수정
- 로그인 사용자에 따라 마스터가 달라지던 문제 해결
- 공동 편집 설정을 마스터만 변경 가능하도록 제한
- 가격 검색 시 부분 매칭으로 엉뚱한 결과가 나오던 문제 해결
- 매장별 비교가 가격 이력과 불일치하던 문제 해결
- 프론트엔드 비밀번호 minLength 6→8 백엔드와 일치
- Safari 호환성 (개발환경 쿠키 secure=false)
- 스캔 에러 시 Gemini 내부 에러 대신 한국어 메시지 표시
- 식재료 수정 시 이름 변경되면 normalized_name 재생성
- 3자 이하 이름은 Gemini 호출 스킵 (API 호출 절약)
- 마스터 탈퇴/이동 시 남은 멤버 중 가장 오래된 사람에게 마스터 자동 이전

## v1.1.0 — 2026-03-30

### 새로운 기능
- **Gemini 영수증 스캔** — 영수증 촬영 한 번에 식재료/가격/매장명 자동 인식. 세제, 생활용품 자동 제외
- **AI 레시피 추천** — 식품안전나라 API + Gemini AI 보충. 냉장고 재료 기반 맞춤 추천
- **가계부** — 월별 지출 바 차트, 식재료별 가격 추이, 매장별 비교, 인플레이션 알림
- **나만의 레시피** — 직접 레시피 등록/수정/삭제, 사진 업로드, 냉장고 재료 매칭 시 추천에 포함
- **가족 구성원 내보내기** — 마스터가 구성원을 내보내기 가능
- **월별 예산 설정** — 예산 초과 시 경고, 냉장고 화면에 프로그레스 바

### 개선
- 직접 등록에 가격/매장명/수량(자유 텍스트) 필드 추가
- 레시피 출처 표시 (나의 레시피 / 식품안전나라 / AI 추천)
- 식재료 이름 정규화 (베지밀B → 두유, 냉동 삼겹살과 생삼겹살 구분)
- 가계부 식재료 검색 자동완성 (구매 이력 기반)

### 버그 수정
- 직접 등록 실패 (quantity 타입 불일치)
- 모달이 네비/헤더에 가려지는 문제
- 레시피 매칭 루프 break 버그
- 등록 시 Gemini 동기 호출로 인한 지연
- 스캔 결과 모달 스크롤 안 되는 문제

### 디자인
- 터치 타겟 44px 최소 보장
- No-Line 규칙 적용 (구분선 제거)
- 섹션 제목 크기 개선
- 스캔 페이지 카메라 영역 축소
