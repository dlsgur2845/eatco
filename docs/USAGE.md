# 사용법

## 회원가입 및 로그인

1. `https://localhost:8443` 접속 → 로그인 화면 표시
2. **"회원가입"** 클릭 → 이름, 이메일, 비밀번호 입력 → 계정 생성
3. 회원가입 완료 시 자동으로 **1인 가족**이 생성되며 냉장고 화면으로 이동
4. 이후 이메일/비밀번호로 로그인

> 하나의 계정은 한 기기에서만 로그인 가능합니다. 다른 기기에서 로그인하면 이전 세션이 만료됩니다.

## 냉장고 (메인 화면)

- **상태 카드**: 오늘 써야 해요 / 3일 이내 / 여유 있어요
- **AI 음식 추천**: 냉장고 재료 기반 맞춤 레시피 + 추가 추천 (출처 표시)
- **인플레이션 알림**: 3개월 전 대비 가격 급등 식재료 알림
- **예산 프로그레스 바**: 이번 달 식재료 지출 현황
- 식재료별 **"일부 사용"** / **"다 썼어요"** 버튼 (undo 지원)

## 영수증 스캔

1. 하단 네비바에서 **"스캔"** 탭 선택
2. **"영수증 촬영하기"** 또는 **"앨범에서 선택"**
3. Gemini AI가 영수증에서 식재료만 추출 (세제, 생활용품 등 자동 제외)
4. 매장명, 가격, 수량도 자동 인식
5. 결과 확인 후 **"냉장고에 추가하기"**

## 직접 등록

1. 하단 네비바에서 **"직접등록"** 탭 선택
2. 우측 상단 **"+ 등록"** 버튼 클릭
3. 등록 모달에서 정보 입력:
   - **상품명**: 입력 시 실시간 자동완성 + 보관기한 정보 자동 표시
   - **보관 방법**: 냉장 / 냉동 / 실온 중 선택
   - **카테고리**: 유제품, 채소/과일, 육류/수산, 가공식품 등
   - **수량**: 자유 입력 (600g, 1L, 3개 등)
   - **가격**: 원 단위 (선택)
   - **구매처**: 매장명 (선택)
   - **유통기한**: 날짜 선택
4. **"등록하기"** 클릭

## 식재료 관리

- **수정**: 식재료 카드를 클릭하면 수정 모달이 열림 (이름, 보관방법, 카테고리, 수량, 가격, 매장, 유통기한)
- **검색**: 상단 검색바에서 식재료명으로 검색
- **필터**: 전체 / 냉장 / 냉동 / 실온 칩으로 필터링
- **삭제**: "선택" 버튼 → 삭제할 항목 선택 → "삭제" 버튼
- **D-Day 표시**: 색상으로 구분 (빨강: D-3 이내, 주황: D-7 이내, 초록: D-8 이상)

## 가계부

1. 하단 네비바에서 **"가계부"** 탭 선택
2. **지출 요약**: 최근 6개월 월별 지출 바 차트
3. **식재료별 추이**: 식재료 검색 (자동완성) → 가격 변동 라인 차트 + 매장별 비교
4. **월 예산 설정**: 예산 입력 → 냉장고 화면에 프로그레스 바 표시

## 알림

- **상단 종 아이콘** 클릭 → 알림 목록
- 유통기한 만료 알림이 자동으로 생성됨
- 안 읽은 알림은 뱃지로 표시

## 알림 주기 설정

1. 하단 네비바에서 **"설정"** 탭 선택
2. 원하는 알림 주기를 체크/해제 (당일, 1일 전, 3일 전 ... 30일 전)

## 가족 관리

설정 → **"가족 관리"** 링크로 접근

- **초대**: 초대 코드 복사 → 가족에게 전달 → 코드 입력으로 참여
- **내보내기**: 마스터가 구성원 옆 "내보내기" 버튼으로 제거 (내보낸 유저는 새 1인 가족 생성)
- **공동 편집**: "공동 편집 허용" 토글 (마스터만 변경 가능)
- **탈퇴**: 2인 이상일 때 탈퇴 가능 (새 1인 가족 자동 생성). 마스터가 탈퇴하면 남은 멤버 중 가장 오래된 사람이 마스터가 됩니다
